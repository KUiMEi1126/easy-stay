# 📱 手机访问指南

## ✅ 问题已解决

### 原因

- ✅ 禁用Radmin VPN虚拟网络
- ✅ 使用以太网IP作为主地址
- ✅ 两个网络都可以访问

## ⚠️ 如果仍无法访问

### 检查步骤

1️⃣ **检查防火墙**
```powershell
# 临时关闭Windows防火墙（测试）
netsh advfirewall set allprofiles state off
```

2️⃣ **检查后端是否运行**
```powershell
Get-NetTCPConnection -LocalPort 3000 -State Listen
```

3️⃣ **检查前端是否运行**
```powershell
Get-NetTCPConnection -LocalPort 5173 -State Listen
```

4️⃣ **在电脑上测试**
```
http://localhost:5173
```

### 如果电脑localhost也无法访问
- 重启后端：`node app.js`
- 重启前端：`npm run dev -- --host`

---

## 🔄 恢复VPN（如需要）

如果之后需要重新启用Radmin VPN：

```powershell
Get-NetAdapter -Name "Radmin VPN" -ErrorAction SilentlyContinue | Enable-NetAdapter
```
