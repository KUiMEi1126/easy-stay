##　本地部署

下载好nodejs后

mobile-client和admin-client前端文件夹用：  npm run dev

然后点击对应网站

server 文件夹用： node app.js  或npm run dev  （start指令加到package了，所以直接start可能有效）
看到终端打印 后端服务已启动: http://localhost:3000 即可。

此时，你访问 http://localhost:3000/api/hotel 应该能看到JSON 数据



同时编写了一键安装依赖和一键启动脚本，直接运行即可



## 服务器部署

1.修改前端的.env.production成服务器地址
2.服务器下载node.js，再用npm安装pm2
3.对于两个前端文件，cd mobile-client
npm run build构建
5.前端用pm2 start start-front.cjs --name easy-stay-frontend
（需要有 start-front.cjs文件）
6.后端用pm2 start app.js --name easy-stay-api

pm2 list  查看服务器
pm2 delete  序号或者all  删除指定序号部署的服务器
