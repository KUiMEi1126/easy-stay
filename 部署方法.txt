更新本地仓库（如果失败先记得改git缓存喵）
# 1. 先拉取最新代码（避免冲突）
git pull

# 2. 确认当前分支
git status
git branch

# 3. 添加修改
git add .

# 4. 提交
git commit -m "你的提交信息"

# 5. 再次确认要推送的分支
git branch -vv

# 6. 推送
git push

# 如果第一次推送，用这个：
git push -u origin 你的分支名
